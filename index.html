Zamena kirillicy na latinicu

Pravila zameny:
1. Mjagkie glasnye: Ja-ja (Яя), Jo-jo (Ёё), Ju-ju (Юю).
2. Dopolnitelǐnye bukvy: Čč (Чч), Ěě (Ээ), Šš (Шш), Žž (Жж), Ǐǐ (Ьь), J̌ǰ (Ъъ).
3. Щ polnostǐju zamenjaetsja bukvoy ш (š):
шаг -> šag;
шея -> шэя -> šěja;
шёпот -> шопот -> šopot;
шик -> шык -> šyk;
шок -> šok;
шутка -> šutka;
щавель - šjavelǐ;
щенок -> шенок -> šenok;
щёлочь -> шёлочь -> šjoločǐ;
щи -> ши -> ši;
щука -> шюка -> šjuka.
4. Ešjo neskolǐko pojasnenij: Cc (Цц), Jj (Йй), Yy (Ыы).
